# Develop

1. `npm i`
2. `npm start`

Open a new terminal:

3. `cd backend`
4. `npm i`
5. `npm start`

# Build & Run in Node.js Environment

1. `npm i`
2. `npm run build`
3. `cd backend`
4. `npm i`
5. `npm start`

# Build & Run for Docker

1. `docker build . -t mas-se-wea/scaffold`
2. `docker run -p 8080:8080 -d mas-se-wea/scaffold`
